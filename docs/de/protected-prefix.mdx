## Geschütztes Präfix (öffentliche IP)

Das **geschützte Präfix** ist der öffentliche IP-Adressblock Ihrer Organisation, den
Cloud gegen DDoS-Angriffe absichert.

| Anforderung | IPv4 | IPv6 |
|-------------|------|------|
| **Mindestgröße** | /24 (256 IPs) | /48 |
| **Routbarkeit** | Öffentlich routbar (nicht RFC 1918) | Globales Unicast (nicht ULA) |
| **Registrierung** | RIR-zugewiesen (ARIN, RIPE, APNIC) | RIR-zugewiesen |
| **Nachweis** | LOA oder IRR/ROA-Registrierung | LOA oder IRR/ROA-Registrierung |

**Datenverkehrsfluss:**

1. Cloud kündigt Ihr Präfix weltweit über BGP Anycast an
1. Der gesamte Internetverkehr, der für Ihre IPs bestimmt ist, wird über Scrubbing-Center geleitet
1. Angriffsdatenverkehr wird verworfen; sauberer Datenverkehr wird über GRE-Tunnel zugestellt
