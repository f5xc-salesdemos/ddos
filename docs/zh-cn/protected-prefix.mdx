## 受保护前缀（公共 IP）

**受保护前缀**是您组织的公共 IP 地址块，Cloud 将针对该地址块提供 DDoS 攻击防御。

| 要求 | IPv4 | IPv6 |
|-------------|------|------|
| **最小规模** | /24（256 个 IP） | /48 |
| **可路由性** | 公共可路由（非 RFC 1918） | 全球单播（非 ULA） |
| **注册** | RIR 分配（ARIN、RIPE、APNIC） | RIR 分配 |
| **证明** | LOA 或 IRR/ROA 注册 | LOA 或 IRR/ROA 注册 |

**流量流向：**

1. Cloud 通过 BGP anycast 在全球范围内宣告您的前缀
1. 所有以您的 IP 为目标的互联网流量均通过清洗中心路由
1. 攻击流量被丢弃；清洁流量通过 GRE 隧道交付
