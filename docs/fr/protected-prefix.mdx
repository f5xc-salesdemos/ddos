## Préfixe protégé (IP publique)

Le **préfixe protégé** est le bloc d'adresses IP publiques de votre organisation que
Cloud défend contre les attaques DDoS.

| Exigence | IPv4 | IPv6 |
|-------------|------|------|
| **Taille minimale** | /24 (256 IPs) | /48 |
| **Routabilité** | Routable publiquement (non-RFC 1918) | Unicast global (non-ULA) |
| **Enregistrement** | Attribué par un RIR (ARIN, RIPE, APNIC) | Attribué par un RIR |
| **Preuve** | LOA ou enregistrement IRR/ROA | LOA ou enregistrement IRR/ROA |

**Flux de trafic :**

1. Cloud annonce votre préfixe globalement via BGP anycast
1. Tout le trafic Internet destiné à vos IPs transite par les centres de filtrage
1. Le trafic d'attaque est rejeté ; le trafic légitime est livré via des tunnels GRE
