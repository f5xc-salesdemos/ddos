## البادئة المحمية (عنوان IP العام)

**البادئة المحمية** هي كتلة عناوين IP العامة الخاصة بمؤسستك التي
يدافع عنها Cloud ضد هجمات DDoS.

| المتطلب | IPv4 | IPv6 |
|-------------|------|------|
| **الحد الأدنى للحجم** | /24 (256 عنوان IP) | /48 |
| **قابلية التوجيه** | قابل للتوجيه العام (غير RFC 1918) | بث أحادي عالمي (غير ULA) |
| **التسجيل** | مُعيَّن من سجل الإنترنت الإقليمي (ARIN، RIPE، APNIC) | مُعيَّن من سجل الإنترنت الإقليمي |
| **الإثبات** | خطاب تفويض (LOA) أو تسجيل IRR/ROA | خطاب تفويض (LOA) أو تسجيل IRR/ROA |

**مسار حركة المرور:**

1. يعلن Cloud عن بادئتك عالمياً عبر BGP anycast
1. يتم توجيه جميع حركة مرور الإنترنت المتجهة إلى عناوين IP الخاصة بك عبر مراكز التنقية
1. يتم إسقاط حركة المرور الهجومية؛ ويتم تسليم حركة المرور النظيفة عبر أنفاق GRE
