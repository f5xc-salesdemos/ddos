## 受保護前綴（公共 IP）

**受保護前綴**是您組織的公共 IP 地址區塊，
Cloud 以此為基礎防禦 DDoS 攻擊。

| 要求 | IPv4 | IPv6 |
|------|------|------|
| **最小大小** | /24（256 個 IP）| /48 |
| **可路由性** | 公開可路由（非 RFC 1918）| 全球單播（非 ULA）|
| **登記** | RIR 分配（ARIN、RIPE、APNIC）| RIR 分配 |
| **證明** | LOA 或 IRR/ROA 登記 | LOA 或 IRR/ROA 登記 |

**流量流向：**

1. Cloud 透過 BGP anycast 在全球宣告您的前綴
1. 所有目的地為您 IP 的網際網路流量均經由清洗中心路由
1. 攻擊流量被丟棄；乾淨流量透過 GRE 隧道傳送
