## 保護プレフィックス（パブリック IP）

**保護プレフィックス**とは、Cloud が DDoS 攻撃から防御する、お客様の組織のパブリック IP アドレスブロックです。

| 要件 | IPv4 | IPv6 |
|-------------|------|------|
| **最小サイズ** | /24（256 IP） | /48 |
| **ルーティング可能性** | パブリックルーティング可能（非 RFC 1918） | グローバルユニキャスト（非 ULA） |
| **登録** | RIR 割り当て（ARIN、RIPE、APNIC） | RIR 割り当て |
| **証明** | LOA または IRR/ROA 登録 | LOA または IRR/ROA 登録 |

**トラフィックフロー：**

1. Cloud が BGP エニーキャストを介してお客様のプレフィックスをグローバルにアナウンスします
1. お客様の IP 宛てのすべてのインターネットトラフィックがスクラビングセンターを経由してルーティングされます
1. 攻撃トラフィックはドロップされ、クリーンなトラフィックが GRE トンネル経由で配信されます
