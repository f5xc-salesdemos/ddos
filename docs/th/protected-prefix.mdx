## Protected prefix (IP สาธารณะ)

**protected prefix** คือบล็อกที่อยู่ IP สาธารณะขององค์กรของคุณที่
Cloud ป้องกันจากการโจมตี DDoS

| ข้อกำหนด | IPv4 | IPv6 |
|-------------|------|------|
| **ขนาดขั้นต่ำ** | /24 (256 IPs) | /48 |
| **ความสามารถในการกำหนดเส้นทาง** | กำหนดเส้นทางได้สาธารณะ (ไม่ใช่ RFC 1918) | Global unicast (ไม่ใช่ ULA) |
| **การลงทะเบียน** | กำหนดโดย RIR (ARIN, RIPE, APNIC) | กำหนดโดย RIR |
| **หลักฐาน** | LOA หรือการลงทะเบียน IRR/ROA | LOA หรือการลงทะเบียน IRR/ROA |

**การไหลของทราฟฟิก:**

1. Cloud ประกาศ prefix ของคุณทั่วโลกผ่าน BGP anycast
1. ทราฟฟิกอินเทอร์เน็ตทั้งหมดที่มุ่งหน้าไปยัง IP ของคุณจะถูกกำหนดเส้นทางผ่านศูนย์กรองการโจมตี
1. ทราฟฟิกที่เป็นการโจมตีจะถูกบล็อก ส่วนทราฟฟิกที่สะอาดจะถูกส่งผ่านอุโมงค์ GRE
