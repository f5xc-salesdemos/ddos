## เอกสารอ้างอิง

### เอกสารอย่างเป็นทางการ

- [L3/L4 Routed DDoS Mitigation Guide](https://docs.cloud.f5.com/docs-v2/ddos-and-transit-services/how-tos/network-firewall/l3l4-ddos-mitigation) — คู่มือทีละขั้นตอนสำหรับการเปิดใช้งานการป้องกัน DDoS แบบ Routed L3/L4 ใน F5 Distributed Cloud
- [K000147949: Initial Routed Configuration Setup](https://my.f5.com/manage/s/article/K000147949) — บทความฐานความรู้ F5 สำหรับรายการตรวจสอบเบื้องต้นของระบบ On-Premises
- [BIG-IP TMOS Routing Administration — Dynamic Routing](https://techdocs.f5.com/kb/en-us/products/big-ip_ltm/manuals/product/big-ip-tmos-routing-administration-14-0-0/11.html) — บทอ้างอิงที่ครอบคลุม BGP4, OSPFv2/v3, IS-IS, RIPv1/v2/RIPng, BFD และ ECMP

### บทความทางเทคนิค

- [Configuring a GRE Tunnel Using BIG-IP](https://community.f5.com/kb/technicalarticles/configuring-a-generic-routing-encapsulation-gre-tunnel-using-big-ip/289030) — คำแนะนำจาก DevCentral สำหรับการสร้าง GRE Tunnel ระหว่างระบบ BIG-IP
- [A Guide to F5 Volumetric (Routed) DDoS Protection](https://community.f5.com/kb/TechnicalArticles/a-guide-to-f5-volumetric-routed-ddos-protection-in-f5-distributed-cloud/341128) — วิธีที่บริการ DDoS แบบ Routed ของ F5 ใช้การวิเคราะห์เชิงพฤติกรรมและศูนย์กรองข้อมูลระดับโลก
- [Protect Your Applications Using F5 Distributed Cloud and Fast ACLs](https://community.f5.com/kb/technicalarticles/protect-your-applications-using-f5's-distributed-cloud-and-fast-acl's/296626) — การกำหนดค่า Fast ACL สำหรับการกรองทราฟฟิกระดับแพ็กเก็ต
- [F5 Distributed Cloud DDoS Auto-Mitigation](https://community.f5.com/kb/technicalarticles/easily-protect-your-applications-from-ddos-with-f5-distributed-cloud-ddos-auto-m/308895) — การเปิดใช้งานการป้องกัน DDoS อัตโนมัติบน HTTP Load Balancer
